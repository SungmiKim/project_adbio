축 · 수산

백신과 항생제 대체가 가능한 애드바이오텍의 Technology를 소개합니다.
축산 수산
SHINVISOL
신비솔
#대사성의약품
닭, 돼지, 소의 식욕촉진, 이뇨작용, 스트레스후의 식욕회복, 모돈회전율 증가, 비육개선, 산란율, 부화율의 개선, 면역력 증진하는 약품

FUNCTION

제품 형태
황색 내지 황갈색의 액체
분류
닭, 돼지, 소의 식욕촉진, 이뇨작용, 스트레스후의 식욕회복, 모돈회전율 증가, 비육개선, 산란율, 부화율의 개선, 면역력 증진
주요 성분
아래의 축종별 용량을 1일량으로 하여 경구 투여
닭 : 음수 또는 사료에 0.1%~0.2% (음수 1L당 본제 1~2ML)로 혼합하여 경구 투여
돼지 : 자돈 : 1일 두당 0.5~1ML 중돈 : 1일 두당 2~5ML 성돈 : 1일 두당 5~10ML
소 : 송아지 : 1일 두당 2~5ML 중송아지 : 1일 두당 6~10ML 젖소, 비육우 : 1일 두당 10~20ML
함량 및 복용방법
100ML, 500ML,1L,5L,10L
수출국
저장방법 : 밀폐된 용기에 넣어 10~25℃의 실온에 보관할 것
유효기간 : 18개월
